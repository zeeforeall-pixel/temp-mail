Agent API

Programmatic API for AI agents and browser automation. Available on window.TempMailAPI

Quick Start
const api = window.TempMailAPI;

// Generate email + wait for OTP in one flow
const session = await api.quickSession();
console.log('Email:', session.address);
const otp = await session.waitForOTP();
console.log('OTP:', otp.otp);
Functions
generateEmail(prefix?, domain?) Create inbox → {address, expires_at}
getMessages(address?) Fetch messages with OTP extracted
waitForOTP(address?, ms?) Poll until OTP arrives (default 60s)
waitForEmail(address?, ms?) Wait for next email + extract OTP
quickSession(prefix?, domain?) Generate email + helper methods
getLatestOTP(address?) Get OTP from loaded messages (sync)
getAllInboxes() List all inbox history
copyEmail() Copy current email to clipboard
